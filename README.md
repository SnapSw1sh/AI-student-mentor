# AI-student-mentor
AI Автонаставник для студентов. Приложение помогает студентам структурировать процесс обучения, получать мгновенные ответы на вопросы и следовать индивидуальному образовательному треку, сформированному ИИ. Клиентская часть платформы для обучения студентов. Реализует интерфейс взаимодействия с ИИ-ментором, визуализацию образовательного трека.
