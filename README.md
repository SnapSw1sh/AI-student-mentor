# AI Student Mentor

**AI Автонаставник для студентов.**
Приложение помогает студентам структурировать процесс обучения, получать мгновенные ответы на вопросы и следовать индивидуальному образовательному треку, сформированному ИИ.

Это клиентская часть (Frontend) платформы, реализующая интерфейс аутентификации и входа в систему.

## Технологии
- **HTML5** — семантическая верстка.
- **CSS3** — адаптивный дизайн (Flexbox, медиа-запросы), нативная стилизация без фреймворков.
- **JavaScript** — клиентская валидация форм, управление состоянием интерфейса (скрытие/показ пароля), имитация отправки данных.

## Как запустить

### Способ 1: Прямой запуск
1. Скачайте проект или клонируйте репозиторий - git clone https://github.com/SnapSw1sh/AI-student-mentor.git
2. Перейдите в папку проекта - cd AI-student-mentor
3. Дважды кликните по файлу **`login.html`**, чтобы открыть его в браузере, либо откройте через проводник.

### Способ 2: Через Live Server (Рекомендуется)
1. Установите расширение **Live Server**.
2. Откройте любой HTML файл проекта (например, `login.html`). Если не открыли, то после `Go Live` Live server предложить открыть любой файл какой хотите.
3. Нажмите `Go Live` в правом нижнем углу редактора.

## Структура проекта

### Страницы
*   **`login.html`** — Страница входа (стартовая).
*   **`register.html`** — Регистрация нового аккаунта.
*   **`reset-password.html`** — Запрос на сброс пароля.
*   **`new-password.html`** — Установка нового пароля.
*   **Служебные страницы:**
    *   `email-sent.html` - страница подтверждения отправки письма
    *   `link-error.html` - страница ошибки ссылки
    *   `register-successful.html` - страница успешной регистрации
    *   `new-pass-successful.html` - страница успешного установки нового пароля

### Ресурсы
*   **`css/style.css`** — Основной файл стилей (включая адаптивность для мобильных устройств).
*   **`js/auth.js`** — Логика валидации форм, обработки ошибок и переключения видимости пароля.

### Примечание
* На некоторые страницы нельзя попасть через другие страницы, для того чтобы попасть на них либо в url введите нужный путь, либо через Live Server.
* Так как тут нет backend, то для некотрых ошибок были добавлены заглушки, которые в последующих версиях уйдут, например для вывода ошибки о том что такой пользователь уже существует на странице регистрации введите "ivan@edu.hse.ru"
